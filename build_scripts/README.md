# Build Scripts

## build.bat
Script para compilar a aplicação Python em um executável Windows (.exe) usando PyInstaller.

### Como usar:
1. Abra o CMD/PowerShell na pasta raiz do projeto
2. Execute: `build_scripts\build.bat`
3. O arquivo `.exe` será criado em `dist/`

### Pré-requisitos:
- Python 3.14+ instalado
- PyInstaller (`pip install PyInstaller`)

### Resultado:
- `dist/Calculadora de Frete.exe` - Executável standalone pronto para distribuição
